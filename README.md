Скачивай все видео с GetCourse на Яндекс Диск **АВТОМАТИЧЕСКИ** через GitHub Actions.

---

## 📦 Что внутри

- `download_final.py` — Главный скрипт (скачивает + загружает)
- `videos.txt` — Файл с видеоссылками (ты заполняешь)
- `.github_workflows_download.yml` — GitHub Actions конфиг (запускает всё в облаке)
- `GITHUB_SETUP.md` — Полная инструкция

## 🕐 Расписание

По умолчанию: **Каждый день в 1:00 AM UTC** (4:00 AM Moscow)

Можешь изменить в `.github/workflows/download.yml`:
```yaml
schedule:
  - cron: '0 1 * * *'  # Измени числа здесь
```

Примеры:
- `0 10 * * *` → 10:00 AM UTC (1:00 PM Moscow)
- `0 0 * * 0` → Каждое воскресенье

---

## 📊 Как работает

1. GitHub запускает скрипт по расписанию
2. Скрипт читает `videos.txt`
3. Скачивает видео через yt-dlp (параллельно по 2)
4. Загружает на Яндекс Диск
5. Сохраняет логи как артефакт

Всё в облаке — твой компьютер не нужен! 💻☁️

---

## 🔐 Безопасность

Токен Яндекс Диска хранится в **GitHub Secrets**, а не в коде.
Это значит он никому не видно, даже если репозиторий публичный.
