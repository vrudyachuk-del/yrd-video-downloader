# GitHub Actions Setup — Пошаговая инструкция

## Шаг 1️⃣ Создать репозиторий на GitHub

1. Открыть https://github.com/new
2. Назвать репозиторий: `yrd-video-downloader`
3. Выбрать **Public** или **Private**
4. Нажать **Create repository**

---

## Шаг 2️⃣ Скопировать файлы в репозиторий

У тебя есть 3 файла:
- `download_final.py`
- `videos.txt`
- `.github_workflows_download.yml`

**На GitHub:**

1. Нажать **Add file → Create new file**
2. Назвать: `.github/workflows/download.yml`
3. Скопировать содержимое из `.github_workflows_download.yml`
4. Нажать **Commit new file**

Повторить для:
- `download_final.py`
- `videos.txt`

---

1. На странице репозитория нажать **Settings**
2. Слева выбрать **Secrets and variables → Actions**
3. Нажать **New repository secret**

6. Нажать **Add secret**

## Шаг 4️⃣ Заполнить videos.txt

1. В репозитории открыть `videos.txt`
2. Нажать карандаш (редактировать)
3. Добавить видеоссылки в формате:
```
НАЗВАНИЕ|ВИДЕОССЫЛКА|КОД_МОДУЛЯ
```
4. Нажать **Commit changes**

---

## Шаг 5️⃣ Запустить автоматизацию

**Вариант 1: Вручную (сейчас)**
1. На странице репозитория нажать **Actions**
2. Слева выбрать **Скачать видео ЯРД 2.0**
3. Нажать **Run workflow**

**Вариант 2: По расписанию**
- Скрипт запускается **каждый день в 1:00 AM UTC** (4:00 AM Moscow)
- Ты можешь изменить время в файле `.github/workflows/download.yml`

---

## Шаг 6️⃣ Смотреть прогресс

1. Открыть репозиторий → **Actions**
2. Выбрать последний запуск
3. Смотреть логи в реальном времени

Когда закончится:
- ✅ Видео будут на Яндекс Диске
- 📝 Логи скачаются как артефакт

---

## ⏰ Расписание

По умолчанию: **Каждый день в 1:00 AM UTC**

Если нужно изменить:
1. Открыть `.github/workflows/download.yml`
2. Найти: `- cron: '0 1 * * *'`
3. Изменить время (формат: `minute hour day month weekday`)

Примеры:
- `0 10 * * *` — 10:00 AM UTC (13:00 Moscow)
- `0 0 * * 0` — Каждое воскресенье в 00:00 UTC
- `0 */6 * * *` — Каждые 6 часов
